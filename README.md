# Cafe Management System

This is a multi-layered full-stack Cafe Management System built with:

## Technologies
- C#, .NET (Web API)
- HTML, SCSS, JavaScript
- Layered Architecture (Business, Data Access, DTO)
- Modular solution structure (.sln)

## Features
- Modular API backend
- Frontend web UI
- Separated business and data access layers
- DTOs for clean API communication

## Architecture
- OM.Business (business logic)
- OM.DataAccess (data access layer)
- OM.Dto (data transfer objects)
- OMApi (REST APIs)
- OMWebUI (web interface)
